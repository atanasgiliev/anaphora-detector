Greek Anaphora & Epiphora Detector

An analytical tool for identifying word and phrase repetition at verse-line boundaries in Ancient Greek texts

Help & Usage

How to use the detector

  1. Upload or paste a UTF-8 Greek text (verse, one line per verse line).
  2. Configure the detection options below.
  3. Click Run Detector.
  4. Results appear below with an annotated text and an occurrence table.

Definitions

Anaphora
Repetition of the same word or phrase at the beginning of consecutive verse lines. Example: πολλοί… / πολλοί… / πολλοί…
Epiphora (Antistrophe)
Repetition at the end of consecutive verse lines.
Symploce
A line may be highlighted for both figures simultaneously (gold start, teal end).

Distance window

The maximum number of consecutive lines within which the repetition must occur. A window of 2 means the same phrase must appear in two immediately adjacent lines; a window of 4 allows up to three intervening lines.

Phrase length

How many words to compare. Length 1 matches single repeated words; length 2 requires both the first (or last) two words to match, and so on.

Stop-word transparency

When enabled, common particles, articles and prepositions (καί, δέ, ὁ, ἡ, γάρ, μέν, ἐν, …) at the start or end of a line are skipped when determining the phrase to compare — but the line is still marked as anaphora/epiphora and the table notes that a stop-word was skipped.

Normalisation options

All options affect only the comparison key; the original text is displayed unchanged in the output.

  • Strip accents — treats ά/α/ᾶ as the same for comparison.
  • Strip breathings — treats ἁ/ἀ as the same.
  • Strip iota subscript — treats ᾧ/ω as the same.
  • Handle elision — strips trailing apostrophe before comparison (ἀλλ' ≈ ἀλλά).
  • Handle nu-movable — strips final ν when preceded by ι or ε (ἐστίν ≈ ἐστί).
